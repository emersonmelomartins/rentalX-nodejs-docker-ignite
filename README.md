## Instalação/Execução
### Instalação via docker

```bash
# docker
docker build -t rentx .
```

### Instalação via node ou yarn
```bash
# yarn
yarn
# node
npm install
```

### Execução do projeto
```bash
# docker
docker run -p 3333:3333 rentx

# docker-compose
docker-compose up

# docker-compose rodando em background
docker-compose up -d

# yarn
yarn dev
# node
npm run dev
```
